## 🚀 How to Use

### View the Website
1. Open `index.html` in any web browser
2. The website is fully responsive and works on desktop, tablet, and mobile

### Test Search Feature
1. Scroll to "Search Resources" section
2. Type keywords (e.g., "Python", "Programming", "Library")
3. Click Search or press Enter
4. View filtered results

### Test Bookmark Feature
1. Scroll to "Featured Resources" section
2. Click "📌 Bookmark" button on any resource
3. Confirmation message appears
4. Bookmarks are saved in browser storage

### Check Browser Console
1. Open DevTools (F12)
2. Go to Application → LocalStorage
3. View saved bookmarks and events
